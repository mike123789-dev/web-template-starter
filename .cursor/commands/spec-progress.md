# Spec Progress

You are reporting current spec-driven progress for this repository.

## Goal

Show a reliable status snapshot for all features and highlight actionable next steps.

## Required Actions

1. Run `npm run specs:check`.
2. Summarize:
   - each feature status (`Draft`, `Ready`, `In Progress`, `Verifying`, `Done`)
   - task progress (`done/total`)
   - blocked tasks
   - needs clarification items
3. Recommend the next feature to start if any feature is `Ready`.

## Output Format

1. Progress snapshot
2. Ready features
3. Blockers and clarification
4. Recommended next action
